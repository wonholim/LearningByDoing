`시나리오` : (HTTP, Ethernet) 사용자의 요청 (GET) 을 서버가 받고, 응답하는 과정

- [ ] 사용자의 입력을 받는 client 생성 및 구현
- [ ] 사용자의 입력을 처리하는 server 생성 및 구현
- [ ] 패킷 구조 구현
- [ ] Application Layer
  - [ ] HTTP 프로토콜 생성 및 구현 
- [ ] Transport Layer
  - [ ] TCP 프로토콜 구현
- [ ] Internet Layer
- [ ] Network Layer
- [ ] 연결 수립을 위한 3-way-handshake 구현
- [ ] 연결 종료를 위한 4-way-handshake 구현
